# La Strada Website

Marketing website for La Strada, a creative and marketing solutions company.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.
